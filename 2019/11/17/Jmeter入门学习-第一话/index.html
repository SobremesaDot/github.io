Jmeter入门学习-第一话

发表于 2019-11-16 | 分类于 性能测试 ， 测试工具

Jmeter的使用范围

接口测试

什么是接口
  • 接口是系统与系统之间进行交互的通道
软件结构
  • UI层
    • 软件的界面显示。由前端人员开发的软件界面
      • 主要使用HTML、CSS、JS等语言进行编写
    • 作用：
      • 界面显示
      • 用户之间的交互（进行操作或查看）
  • 业务逻辑层（接口层）
    • 后端人员进行逻辑运算（使用Python、JAVA、C、C++等语言编写）
    • 作用：进行业务逻辑处理与运算
  • 数据层
    • 为借口提供数据的查询、修改等

性能测试

B/S架构
  • 关注的重点在服务器，包括以下三点：
    • 最大并发数：服务器能够承受的用户并发数量
    • 吞吐量：服务器每秒处理的数据量大小
    • 响应时间（从客户端——服务器的整个过程所需要的时间）
C/S架构
  • 内存占用
  • CPU占用
  • 启动时间
  • 切换时间
  • 等

Jmeter常用功能

线程组

  • 用户管理测试中的虚拟用户
  • 线程数 = 虚拟用户数
  • Ramp - up Period = 执行时间
  • 循环次数
    • 单个用户的循环次数
    • 永远：一直执行

脚本录制

BadBoy录制
  • HTTP Cookie Manager：用于管理当前场景产生的Cookie
  • User Defined Variables：用于管理当前场景产生的Session
  • HTTP Header Manager：用于管理请求消息头
    • 请求方式
    • 长连接
    • 客户端编码格式
    • 压缩方式等
  • 缺点：
    • 不能录制不兼容IE浏览器的网站
    • 自动过滤资源类文件（JS、图片文件、CSS）
HTTP代理服务器录制
  • 首先清理浏览器缓存（ctrl+shift+delete该命令所有浏览器通用）
  • 采用包含模式录制：只录制某一类请求（php / js / jpg / png等）
  • 排除模式（过滤模式）：不录制某一类请求
    • 书写格式：.*\.jpg.*，.*\.js.*

参数化

  • 便于测试数据的管理、维护和使用
  • 设置用户变量
  • 外部文件参数化：CSV
    • filename：要引用的外部文件的路径和文件
    • file encoding：文件的编码格式（utf-8）
    • variable names：变量的名字，多个变量通过逗号隔开
    • deLimiter：文件中的数据的分割方式
      • 空格为分隔符
      • tab键为分隔符（\t）
      • ,为分隔符（默认的分割方式）
    • allow quoted data：是否支持全角输入
    • Recycle on EOF：是否支持循环
    • stop thread on EOF：测试数据使用完，就停止执行
    • sharing mode：测试数据的有效范围
  • 随机数：可以使用随机数来生成大量不重复的测试数据
    • ${__Random(100000,999999,)}

断言

  • 断言是测试的预期结果
  • 响应断言
    • 判断响应信息中是否包含指定关键字
    • 查找范围
      • 响应文本：检查页面返回源代码
      • Document：检查程序处理之前的源代码
      • URL地址：检查请求的URL地址
      • 响应代码：
        • 2XX：请求成功
        • 3XX：请求重定向
        • 4XX：客户端错误
        • 5XX：服务器错误
      • 响应信息
        • OK：响应成功
        • not found：找不到页面
      • Response Header：响应数据的消息头
    • 匹配规则
      • 包括（模糊查询）
      • 匹配（精确查询）
      • Equals
      • Substring
      • 否
  • 时间断言：判断单个请求的时长
  • 大小断言：判断单个请求返回数据的大小

时间控制器

  • 固定定时器的作用范围
    • 如果定时器存放在目录中，对目录中的所有请求生效
    • 如果定时器存在某个请求中，只对单个请求生效
  • 随机定时器：作用范围与固定定时器相同
  • 随机定时器
  • 集合点：可以通过设置集合点，来达到并发测试的目的

逻辑控制器

  • 循环控制器：用户控制循环次数
  • 随机控制器

结果查看

  • 查看结果树
  • 图形报告
  • 聚合报告中的指标
    • average
    • median
    • 90%line
    • min
    • max
    • error
    • throughput
    • 流量
